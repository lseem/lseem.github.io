Introduction

Definitions

Let \(C_\bullet\) and \(D_\bullet\) be chain complexes. A *chain map* from \(C_\bullet\) to \(D_\bullet\) is a collection of homomorphisms \( f_n:C_n\to D_n \) such that \(d_nf_n=f_{n-1}d_n:C_n\to D_{n-1}\) for all \(n\in\mathbb{Z}\).

In other words, a chain map is a collection of homomorphisms such that each square in the above diagram commutes.

Lemmata

Lemma: A chain map \(f_\bullet:C_\bullet\to D_\bullet\) induces homomorphisms \(f_n:H_n(C_\bullet)\to H_n(D_\bullet)\) in homology, for all \(n\in\mathbb{Z}\).

Proof: Think about it for a moment; any element in \(H_n(C_\bullet)\) can be represented by \(\sigma + {\rm im}(d_{n+1}^C)\) where \(\sigma\in C_n\). Then the map \[ \sigma + {\rm im}(d_{n+1}^C) \in H_n(C_\bullet)\overset{f_*}{\longrightarrow} H_n(D_\bullet)\ni f(\sigma) + {\rm im}(d_{n+1}^D) \] is a homomorphism - you can check it for yourself. This is exactly the induced homomorphism \(f_*:H_n(C_\bullet)\to H_n(D_\bullet)\).

QED.
